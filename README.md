# CRUD em Java

Este repositório foi desenvolvido com o objetivo de praticar conceitos de programação back-end utilizando Java e aprofundar meus conhecimentos em Git e GitHub.

O projeto consiste em um CRUD (Create, Read, Update e Delete) desenvolvido em Java 25, utilizando arrays como armazenamento temporário de dados.

## Funcionalidades
- Criar usuários
- Listar usuários
- Atualizar usuários
- Deletar usuários

## Tecnologias utilizadas
- Java 25
- Git
- GitHub

## Objetivo do projeto
Praticar:
- Lógica de programação
- Estruturação de código
- Manipulação de dados
- Organização de projetos Java
- Controle de versão com Git e GitHub
